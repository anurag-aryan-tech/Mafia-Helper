# Mafia Helper

Mafia Helper is a desktop application built using Python's Tkinter library, designed to facilitate the setup and management of the popular social deduction game, Mafia. This application streamlines the process of assigning roles, managing players, and ensuring a smooth game flow.

## Features
- **Player Management**: Easily add and manage players with their respective roles.
- **Role Assignment**: Assign roles such as Mafia, Sheriff, Doctor, and Villager with built-in constraints to ensure game balance.
- **Responsive UI**: Utilizes CustomTkinter for a modern and responsive user interface.
- **Zoom and Resize**: Convenient window management with zoom and resize functionalities.

## Installation

### Prerequisites
- Python 3.10+
- pip (Python package manager)

### Setup
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Mafia-Helper
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. Run the application:
   ```bash
   python Helper.py
   ```
2. Follow the on-screen instructions to set up your game.
3. Use the main menu to navigate through different windows for player and role management.
